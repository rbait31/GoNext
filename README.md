# GoNext
Дневник туриста

Приложение для планирования поездок и ведения дневника посещённых мест.

## Возможности

- **Места** — база мест с координатами, фотографиями, чекбоксами «Посетить позже» и «Понравилось»
- **Поездки** — маршруты с упорядоченными местами; режимы «План» и «Дневник»
- **Следующее место** — первый непосещённый пункт активной поездки
- **Карты** — выбор координат на карте, геолокация, открытие в навигаторе
- **Фотографии** — съёмка/выбор из галереи, привязка к местам и пунктам маршрута, полноэкранный просмотр
- **Настройки** — тёмная тема, оттенок темы, язык (ru/en), о приложении (версия и описание)

## Запуск

```powershell
npx expo start
```

## Структура

- `app/` — экраны (Expo Router)
- `lib/` — DAL, БД, сервисы (фото, геолокация, координаты)
- `assets/backgrounds/` — фоновые изображения экранов (`bg-home.png`)
