# PLAN.md
## Поэтапный план реализации приложения GoNext

---

## Этап 0. Подготовка инфраструктуры

- [x] 1. Установить и настроить SQLite для React Native (expo-sqlite)
- [x] 2. Создать базовую структуру папок: `lib/`, `components/`, `hooks/`
- [x] 3. Настроить типы TypeScript для сущностей (Place, Trip, TripPlace)
- [x] 4. Создать схему БД и скрипты миграции
- [x] 5. Реализовать слой доступа к данным (data layer / DAL)

---

## Этап 1. База данных

- [x] 1. Спроектировать таблицы:
   - `places` (id, name, description, visitlater, liked, lat, lng, createdAt)
   - `trips` (id, title, description, startDate, endDate, createdAt, current)
   - `trip_places` (id, tripId, placeId, order, visited, visitDate, notes)
   - `place_photos` (id, placeId, filePath)
   - `trip_place_photos` (id, tripPlaceId, filePath)
- [x] 2. Реализовать инициализацию БД при первом запуске
- [x] 3. Реализовать CRUD-операции для Place
- [x] 4. Реализовать CRUD-операции для Trip и TripPlace

---

## Этап 2. Главный экран и навигация

- [x] 1. Переработать текущий экран Home в главное меню:
   - Заголовок: GoNext
   - Кнопки: Места, Поездки, Следующее место, Настройки
- [x] 2. Настроить маршруты Expo Router:
   - `/` — главное меню
   - `/places` — список мест
   - `/places/[id]` — карточка места
   - `/places/[id]/edit` — редактирование места
   - `/places/new` — создание места
   - `/trips` — список поездок
   - `/trips/new` — создание поездки
   - `/trips/[id]` — детали поездки (маршрут)
   - `/next` — следующее место
   - `/settings` — настройки
- [x] 3. Оформить навигацию компонентами React Native Paper (FAB, List, Card, etc.)
