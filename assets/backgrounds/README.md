# Фоновые изображения

Папка для фоновых изображений экранов приложения.

**Текущий файл:** `bg-home.png` — placeholder (можно заменить на своё изображение).

Положите сюда свой файл с именем `bg-home.png` (или `.jpg` — тогда измените расширение в `lib/backgroundAsset.ts`). Изображение отображается на всех экранах приложения.
